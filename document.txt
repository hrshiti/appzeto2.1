# Appzeto – 3D Animated MERN Stack Website

## 1. Project Overview

**Project Name:** Appzeto Official Website
**Company Type:** Software Development Company
**Goal:** Build a modern, fully responsive, 3D animated corporate website that showcases Appzeto’s services, products, technologies, projects, partners, and reviews, while generating leads and career applications.

**Primary Objectives:**

* Establish a strong digital brand presence
* Showcase products, services, and technical expertise
* Generate leads via Contact & Reviews forms
* Attract talent via Careers CTA

---

## 2. Tech Stack

### Frontend

* **React.js** (Vite)
* **Tailwind CSS** (base styling)
* **UI Libraries (selective use):**

  * shadcn/ui (preferred for consistency)
  * Material UI / Ant Design (only where required)
  * Mantine / Chakra UI (optional)

### Animations & UX

* **GSAP** – advanced animations, timelines, scroll triggers
* **Framer Motion** – component-level animations
* **Lenis** – smooth scrolling
* **Three.js / React Three Fiber (optional)** – 3D hero elements

### Backend

* **Node.js + Express.js**
* **MongoDB** (Mongoose)

### Other

* Responsive design (Mobile-first)
* SEO-ready structure
* Optimized images & lazy loading

## 4. Homepage Section-wise Detailed Plan

### 4.1 Hero Section (Above the Fold)

**Purpose:** First impression + Brand positioning

**Content:**

* High-quality 3D/illustrated image or abstract tech visual
* Company tagline (bold, modern typography)
* Short description (1–2 lines)
* CTA Buttons:

  * "Contact Us"
  * "Careers"

**Design & Animation:**

* GSAP text reveal animation
* Framer Motion button hover effects
* Subtle 3D parallax on image
* Smooth scroll indicator

---

### 4.2 About Us Section

**Purpose:** Build trust & credibility

**Content:**
